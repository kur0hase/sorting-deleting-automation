### Function summary:
- Naming and enumerating files (of the same file type)
- Sorting/moving it to the folder of the same type
- Remove unpreferred file type in bulk

### How to use:
- Make sure [Python](https://www.python.org/downloads/) is installed
- Git clone this repository or download the **main-script.py** file
- Open your terminal, type ```python main-script.py```
- The script will run and shows options and is ready to use
